# ✨ Aurum Dining — Premium Restaurant Website

##Link: https://rishwebb.github.io/Restaurant.website/

Welcome to a complete premium restaurant website experience designed with an elegant dark-luxury visual language, smooth animations, and full multi-page flow.

## 🌟 Experience Highlights

- **Home Page** with hero storytelling, featured signatures, metrics, and newsletter subscription.
- **Menu Page** with full categorized menu and quick add-to-cart actions.
- **Products Page** with premium product cards and direct order/cart actions.
- **Order Page** with complete customer order form, quantity handling, and dynamic total.
- **Cart Page** with editable quantities, GST calculation, grand total, remove and clear actions.
- **About Us Page** with brand story and value proposition.
- **Contact Us Page** with complete contact details and enquiry form.
- **Smart Chatbot** fixed at bottom-right with pre-filled FAQs and instant answers.
- **Elegant UI System** with responsive layout, polished cards, gradients, and motion effects.

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom premium theme and animations)
- Vanilla JavaScript (full page behavior, cart system, chatbot, forms)

## ✅ Functional Scope Delivered

- Full navigation across all required pages.
- LocalStorage-powered cart persistence.
- Dynamic data rendering across menu/products/home sections.
- End-to-end add-to-cart + order + cart pricing logic.
- Interactive chatbot with curated support answers.
